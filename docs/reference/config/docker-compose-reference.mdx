# Docker Compose Reference

## Base sandbox compose

`docker-compose.yml` is the public local sandbox and observability stack.

Core services include:

- `astra-node1`, `astra-node2`, `astra-node3`
- `minio`, `minio-init`
- `traffic-gen`
- `prometheus`, `grafana`

## Production-adjacent compose assets

The repo also ships compose overlays for the Astra-backed single-node K3s path:

- `refs/scripts/deploy/docker-compose.k3s-single-node.production.yml`
- `refs/scripts/deploy/docker-compose.k3s-single-node.object-store.yml`
- `refs/scripts/validation/docker-compose.image.yml`

Important rule:

- the validation compose defaults are benchmark-oriented,
- the production single-node overlays are the right starting point for real K3s+Astra deployments.

For parameter-level runtime tuning, use [/reference/config/env-vars](/reference/config/env-vars).
