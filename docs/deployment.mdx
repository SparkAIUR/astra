## Deployment rules that matter

- keep explicit resource budgets for Astra in production,
- treat benchmark compose defaults as test profiles, not operating defaults,
- make boot ordering explicit when K3s depends on Astra locally,
- keep object-tier or backup configuration deliberate rather than accidental,
- validate control-plane add-ons against Astra memory and read-path behavior before rolling them broadly.
