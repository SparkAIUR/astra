# FAQ

## Is Astra a full etcd replacement?

Astra is etcd-compatible for the validated control-plane scenario set and continues to expand parity through phase validation and regression gating.

## Why does Astra emphasize queue-stage telemetry?

Aggregate p99 alone cannot isolate contention domains. Stage telemetry distinguishes enqueue pressure from quorum commit latency.

## Can I run multiple tenants in one Astra cluster?

Yes. Tenant virtualization uses prefixed keyspaces and auth claim mapping to isolate overlapping keys.

## How do I evaluate Astra quickly?

Use `quickstart.sh`, run non-smoke validation scripts, and compare against your etcd baseline with matched workload shape.
