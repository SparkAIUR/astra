# Memory and Backpressure

Astra uses bounded memory tracking and sheds load before OOM.

## Pressure Bands

- Normal
- High
- Critical

## Implications

- write admission may be delayed or rejected under critical pressure,
- validation harnesses should include memory-pressure scenarios for candidate profiles.
