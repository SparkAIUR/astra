# Tiering and S3

Astra writes compacted chunks plus manifest metadata to object storage.

## Data Model

- Chunk objects with metadata and CRC32C.
- `manifest.json` references chunk set and revisions.

## Restore Model

`astrad` can recover from local manifests and `s3://` manifest sources for bulk-load workflows.
